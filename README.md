# Studio Web Framework

A responsive static website framework for creative studios, featuring modular pages for portfolios, about sections, news, and contact, built with HTML, CSS, and JavaScript for easy customization and deployment.

## Features

- **Modular Design**: Reusable modules in the HTML files allow for easy replication and customization of sections like image sliders, navigation, and content blocks.
- **Responsive Layout**: Optimized for various screen sizes with clean, modern styling.
- **Interactive Elements**: Includes JavaScript-powered sliders, tab switchers, and keyboard navigation.
- **Lightweight**: No external dependencies, pure HTML/CSS/JS for fast loading.

## Structure

- `index.html`: Homepage
- `about.html`: About page with links to detail topics
- `works.html`: Portfolio/projects page
- `news.html`: News/events page
- `contact.html`: Contact information
- `about/`: Detail pages for about topics (topic-a.html, etc.)
- `works/`: Individual project pages (works-001.html, etc.)
- `assets/css/`: Stylesheets (base, components, layout, pages)
- `assets/js/main.js`: JavaScript for interactivity
- `media_doc/`: Placeholder media files

## Usage

1. Clone or download the repository.
2. Customize the HTML files, replacing placeholders with your content.
3. Update images in `media_doc/` and links as needed.
4. Open `index.html` in a browser to preview.

## Customization

- Edit CSS in `assets/css/` for styling changes.
- Modify JavaScript in `assets/js/main.js` for behavior adjustments.
- Replicate reusable modules (e.g., sliders, tabs) across pages by copying HTML structures.
